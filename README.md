# Food-Nutrition-Analytics-ML

## Описание проекта

В рамках проекта разработана система анализа и оценки блюд на основе их ингредиентов.

Цель — определить полезность и потенциальную “качество” блюда, а также предоставить пользователю информацию о нутриентах и рекомендации похожих рецептов.

---

## Задачи проекта

- Классификация блюд по уровню качества (`bad`, `so-so`, `great`)  
- Расчет нутриентного состава ингредиентов  
- Определение процента суточной нормы нутриентов  
- Подбор похожих рецептов  

---

## Данные

В проекте использовались:

- датасет рецептов (состав ингредиентов и рейтинг)  
- данные о нутриентах  
- ссылки на рецепты  

Данные были предварительно очищены и отфильтрованы, оставлены только релевантные признаки (ингредиенты и нутриенты).

---

## Методология

### 1. Подготовка данных

- очистка датасета  
- отбор признаков (ингредиенты, нутриенты)  
- формирование обучающей выборки  

---

### 2. Моделирование

Рассмотрены два подхода:

#### 🔹 Регрессия
- предсказание численного рейтинга  
- сравнение моделей по метрике RMSE  

#### 🔹 Классификация
- преобразование рейтингов в классы  
- `bad`, `so-so`, `great`  

- обучение моделей классификации  
- подбор гиперпараметров (GridSearchCV)  
- сравнение с baseline  

---

### 3. Feature Engineering

- использование нутриентов как признаков (~20 признаков)  
- формирование признакового пространства для моделей  

---

### 4. Оценка моделей

Используемые метрики:

- accuracy  
- precision  
- baseline (naive classifier)  

---

## Результаты

- accuracy: **0.60**  
- precision: **0.70**  
- baseline: **0.55**  

Модель показала улучшение по сравнению с базовым подходом.

---

## 🍽 Функциональность системы

Приложение принимает список ингредиентов и:

### 1. Прогнозирует класс блюда
- bad / so-so / great  

---

### 2. Выводит нутриенты

Для каждого ингредиента:
- белки  
- жиры  
- углеводы  
- микроэлементы  

+ процент от суточной нормы  

---

### 3. Рекомендует рецепты

- находит **3 наиболее похожих рецепта**  
- выводит:
  - название  
  - рейтинг  
  - ссылку
